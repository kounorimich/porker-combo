# model

#1. CardSuit

   * Club: 'c',
   * Diamond: 'd',
   * Heart: 'h',
   * Spade: 's'


#2. CardVNumber
   * 2 ~ 14までのカードの数字
     * Aは一番強いため、1ではなく14で表現


#3. Card
   * 52種類 (13 × 4) 
      Aクラブ、Aダイヤ、Aスペード, Aハート、Kクラブ、Kダイヤ................2ダイヤ

#4. Combo
  * 2枚のカードの組み合わせ
  * 例
    * Aクラブ、Aダイヤ
    * Aダイヤ、Aスペード
    * Aクラブ、Kクラブ

#5 Hand
  * 「Aのポケットペア」「AとKのスーテッド」「AとKのオフスーテッド」といった、Comboを戦略的に同一のものでまとめたもの。
  * 「Aクラブ、Aダイヤ」「Aダイヤ、Aスペード」といったComboは、「Aのポケットペア」というHandに所属する。
